## Project Setup
1. Clone the repository
2. Navigation into the project directory and install the project depencies with:
 NPM
  ```bash
  npm install
  ```
  or with yarn
  ```bash
  yarn
  ```
3. Setup the back url endpoint in `lib/config/index.js`
4. Run the project with
   ```bash
   npm run start
   ```
   or
   ```bash
   yarn start
   ```
